# Open Problems

These are the highest-value unresolved problems visible in the current project
materials.

## 1. Ecologies That Select For Richer Cognition

The project still lacks a clearly validated ecological recipe that reliably
selects for memory, planning, navigation, and adaptive internal state instead
of reactive local strategies.

## 2. Diversity Preservation

The system already uses champion carryover and fresh injections, but it does
not yet have a strong speciation or niche-management story. This leaves open
the risk of premature convergence and degenerate dominance.

## 3. Better Cognitive Readouts

The evaluation harness is much stronger than single-score benchmarking, but the
lab still needs better measures of meaningful cognition and open-ended progress.

## 4. Credit Assignment Across Timescales

The project mixes immediate reward, within-lifetime learning, and lineage-level
selection. It is still unclear how these interact and where the main leverage
is.

## 5. Modularity, Hierarchy, And Reuse

If the long-term goal is sustained complexification, the system may eventually
need stronger mechanisms for modular structure and reusable subcircuits.
