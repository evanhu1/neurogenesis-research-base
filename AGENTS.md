# Agent Instructions

This repository is a knowledge base for an ongoing neuroevolution research
project. The purpose of this repo is to support the scientific method and help
the lab do original, novel science research.

Start by reading `research_statement.md`, then read `index.md`.

When contributing here, optimize for clarity, traceability to sources, and
research usefulness. Prefer explicit hypotheses, concrete evidence, and
carefully separated observations, interpretations, and open questions.
